SAGA Counters are a special category of Hadoop counters because they are not present natively in Hadoop. The values of these counters are calculated from the SAGA framework. These counters allow to track the number of objects processed by each scientific module and thus to detect abnormal behavior (inadequate filtering, duplicate data, etc.).

Calculation

'In' value of SAGA counters is the sum of 'HDFS-In' and 'Gbin-In' counters name for a given job.
'Out' value of SAGA counters is the sum of 'HDFS-Out' and 'Gbin-Out' counters name for a given job.

Example

For this job:
  • in = 59 + 59 + 59 + 59 + 59 = 295
  • out = 59 + 7 + 150 + 30 + 59 = 305
